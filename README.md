# ❤️ Heart Disease Prediction

## 📖 Description
Built a machine learning model to predict heart disease using RandomForestClassifier. Performed data preprocessing, feature engineering, and data cleaning using Pandas and NumPy. Trained and evaluated the model to improve prediction accuracy.

---

## 🚀 Features
- Data preprocessing and cleaning  
- Feature engineering  
- Machine learning model using RandomForestClassifier  
- Model evaluation for accuracy  

---

## 🛠️ Technologies Used
Python, Pandas, NumPy, Scikit-learn, RandomForestClassifier

---

## 📂 Dataset
The dataset contains medical attributes such as:
- Age  
- Sex  
- Blood Pressure  
- Cholesterol  
- Heart Rate  
- Other clinical parameters
